# Graduation Platform 🎓

Graduation Platform is a modern web application designed to celebrate graduates by creating personalized pages where friends can share wishes, videos, and memories.

The platform also includes a developer CV builder to help graduates create and download a professional resume.

## Features

- Personal graduation pages for each graduate
- Photo gallery for university memories
- Graduation video section
- Friends wish system (text and video messages)
- Developer CV builder with PDF export
- Private dashboard for graduates to manage their content
- Confetti celebration animations

## Tech Stack

Frontend:
- Next.js
- Tailwind CSS
- shadcn/ui

Backend:
- Supabase (PostgreSQL + Storage)

Libraries:
- React Hook Form
- Zod
- Framer Motion
- Chart.js
- React PDF
- Canvas Confetti

Deployment:
- Vercel

## Purpose

This project is built as both a graduation celebration platform and a full-stack portfolio project demonstrating modern web development using Next.js and Supabase.
